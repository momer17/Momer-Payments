# Payment Platform

A distributed payment processing platform that provides secure, scalable payment services with Verification of Payee (VoP) capabilities, saga-based orchestration for transaction consistency, and event-driven messaging through RabbitMQ. The platform ensures reliable payment processing across multiple microservices while maintaining data integrity and providing real-time transaction status updates.
